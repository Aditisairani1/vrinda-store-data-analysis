# Vrinda Store Sales Data Analysis 📊

## 📌 Project Overview

This project analyzes sales data from Vrinda Store to identify trends in customer behavior, product performance, and revenue generation. The analysis supports data-driven decision-making.

## 🎯 Objective

* Analyze sales trends
* Identify top-performing products
* Understand customer demographics
* Improve business decisions

## 🛠 Tools Used

* Microsoft Excel (Pivot Tables, Charts, Data Cleaning)

## 📊 Key Analysis Performed

* Sales by State
* Sales by Gender & Age
* Sales by Channel
* Monthly Sales Trends

## 🔍 Key Insights

* Majority of sales come from specific states and customer segments
* A few products contribute to most of the revenue
* Sales vary across different channels
